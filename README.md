# CodeHarbor

**CodeHarbor** — Full-Stack Learning Management System | *Next.js 14, TypeScript, Prisma, PostgreSQL, LiveKit, Socket.io*

- Engineered a full-stack LMS with Next.js 14 (App Router), TypeScript, and Prisma/PostgreSQL, featuring multi-role access control for Students, Instructors, Parents, and Admins with distinct dashboards and permissions
- Integrated real-time video (LiveKit), live messaging (Socket.io), Google OAuth (NextAuth.js), file uploads (UploadThing), and rich content editing with drag-and-drop course management using Zustand and TanStack Query
